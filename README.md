# SquareGame

SquareGame — RPG игра с квадратиком, который бегает и стреляет во врагов — других квадратиков.

### Функциональность

* Меню
* Экран смерти
* Здоровье игрока
* Статистика в JSON
* Перемещение на WASD
* Несколько видов оружия
* Выстрел в сторону курсора
* Генератор рандомных уровней
* Враги преследуют и стреляют в игрока
